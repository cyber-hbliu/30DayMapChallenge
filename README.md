# 30DayMapChallenge

## #1: Points

I mapped the checkin points under hashtag "ustrip" in North America on Instagram.The data is from [Voratham Tiabrat](https://zenodo.org/records/3530864#.Y9Y5itJBwUE), including the records during Jan 2016 to Oct 2019. I mapped it with R and modified in Adobe Illustrator.
